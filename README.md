# Day 1 - Count Even Numbers

## Problem
Count the even numbers in a list.

## Algorithm
1. Create count = 0.
2. Visit each number.
3. If it is even, increase count.
4. Print count.

## Concepts Used
- List
- for loop
- if statement
- Variables

## What I Learned
Today I learned how to count items in a list using a for loop and an if statement.
